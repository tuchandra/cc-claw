# CC Claw

A solver for the "Claw. Grab. Prize!" arcade game. Enter your guesses and shake counts, and it tells you what to try next.

**Features**
- Suggests the optimal next guess (minimax strategy)
- Preview what each possible shake outcome means for the suggested guess
- Click any remaining possibility to explore its shake breakdown

**Tech**
- Bun + TypeScript + React + Tailwind CSS
- Deployed to GitHub Pages via GitHub Actions
